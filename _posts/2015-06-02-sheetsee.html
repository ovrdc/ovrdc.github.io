--- layout: maps description: This map uses the sheetsee.js library to load a GeoJson from a Google Spreadsheet. The sheet is linked to a Google Form, which automatically enters the lat/long into the form response when opened from the GPS Form map. It is limited to accepting coordinates within a small area around Ohio. The map seems to work best when there is only one sheet in the spreadsheet. The data is loaded into the console, so you can check that if you wish. I am using the basic Leaflet code to build the map, not the sheetsee code, so as to better integrate this method into other Leaflet maps. --- Sheetsee Map Example
